Cara kerja import:
• Kolom NOGARDU dengan suffix (-1, -K dll) di-strip otomatis ke kode gardu induk.
• Klik 🔍 Cek Duplikat DB sebelum import — menandai baris yang sudah ada (SKIP).
• Duplikat dicegah di level DB via ON CONFLICT DO NOTHING berdasarkan (no_gardu + tgl_ukur + jam_ukur + petugas).
• File .xls / .xlsx PLN langsung didukung, tidak perlu convert ke CSV.
